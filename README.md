# Dashboard Ana - Gestão Financeira

Aplicação Streamlit para gestão financeira pessoal com controle de receitas, despesas e antecipação de parcelas.

## Funcionalidades

- 📊 **Gestão Executiva**: Dashboard com resumo mensal de receitas, despesas e saldo líquido
- 📅 **Detalhamento Mensal**: Visualização detalhada por mês
- 🛠️ **Gerenciar Itens**: CRUD de itens financeiros
- ⚡ **Antecipar Parcelas**: Sistema de antecipação de recebíveis

## Deploy

Este projeto está configurado para deploy automático na Vercel.

## Tecnologias

- Streamlit
- Pandas
- Plotly
- Python
